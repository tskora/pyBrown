# pyBD

Brownian and Stokesian dynamics simulation software.

## Units

| Physical property | Units |
|---|---|
| Temperature | kelvin (*K*) |
| Viscosity | poise (*P*) |
| Time | picosecond (*ps*) |
| Distance | angstrom (*Å*) |
| Energy | kilocalorie per mol (*kcal/mol*) |
| Energy | kilocalorie per mol per angstrom (*kcal/mol/Å*) |
